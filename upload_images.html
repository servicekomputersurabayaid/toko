Kelola Gambar

1. Upload Gambar Masal

Pilih banyak gambar sekaligus, lalu klik Upload. URL akan muncul di bawah.


Jika diisi, gambar akan direname menjadi nama-seo-[random].jpg.



Hasil URL (Format CSV):

Copy hasil di bawah ini ke kolom image_url di file CSV Anda.

2. Hapus Gambar via URL

Paste URL gambar (Firebase Storage) di sini untuk menghapusnya secara permanen. Bisa banyak sekaligus (pisahkan dengan Enter atau Koma).

Preview gambar akan muncul di sini...

3. Rename Gambar Masal via CSV

Gunakan ini jika gambar terlalu banyak untuk dimuat di galeri. Upload CSV dengan format: URL_LAMA, NAMA_BARU (tanpa header).

Contoh isi CSV:
https://firebasestorage.../img1.jpg, laptop-asus-rog
https://firebasestorage.../img2.jpg, mouse-logitech

4. Galeri Penyimpanan (Folder Products)

Lihat dan hapus gambar yang sudah tidak terpakai di server.
